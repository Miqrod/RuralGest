# 🧩 Módulo: Eventos

## 🎯 Propósito

Los eventos son la fuente única de verdad del sistema.

## 🧠 Responsabilidades

* registrar cambios en el sistema
* mantener trazabilidad
* servir de base para estados derivados

## 🔗 Entidades relacionadas

* animal
* lote
* venta_linea

## ⚙️ Reglas clave

* los eventos son inmutables
* no se pueden editar
* toda modificación se realiza mediante nuevos eventos

## 🚀 Casos de uso

* destete
* venta
* muerte
